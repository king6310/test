快捷键

复制一行 shift+alt+↓
选定多个相同的单词 ctrl+D
添加多个光标 ctrl+alt+↑(↓)
添加光标到所有实例中 Ctrl + Shift + L
选择当前行 Ctrl + L
全局替换某个单词 ctrl+h
快速定位到某一行 ctrl+g
选择某个区块 shift+alt
放大或缩小编辑器 ctrl +/-
代码折叠 Ctrl + Shift + [/]

插件

1. Open-in-browser 浏览器打开html文件
2. Quokka 即时预览函数或表达式结果
3. Faker 随机生成姓名、地址等信息
4. CSS Peek 预览或跳转到对应CSS样式定义
5. TODO-Highlight 高亮标记TODO / FIXME标签
6. Chinese (Simplified) Language Pack for Visual Studio Code（推荐） 官方IDE汉化包
7. 翻译（英汉词典） 支持对驼峰/下划线命名的变量名翻译
8. Code Spell Checker 英文单词拼写检查
9. Darcula Theme 深色IDE主题
10. Bracket-pair colorizer 2（推荐） 彩色括号以及配对
11. GitLens（推荐） 增强自带Git功能
12. Path intellisense 文件路径智能补全
13. VSCode Icons 官方图标美化包
14. npm Intellisense 写js时添加npm依赖包提示
15. Settings Sync 同步设置到github（VSCode其实自带同步）
